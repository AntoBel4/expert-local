# Methode Video Expert Local - Style Remotion

## 1. DECOUPAGE SCENES (12 videos courtes)

Au lieu d'une video longue, faire des chunks de 5-8 secondes chacun.

---

## 2. PROMPTS KILO (SVG/Images style Remotion)

### Scene 1 (0-5s) - Intro logo
```
Background noir absolu. Logo "Expert Local" en blanc avec glow subtle. Animation: le logo apparait en fade-in avec un léger scale. Style: minimaliste, typographie moderne, glow neon blanc. Format: SVG ou PNG transparent.
```

### Scene 2 (6-12s) - Rue commerçante
```
Illustration vectorielle plate d'une rue de ville française (Chartres). Style: colors=#1E293B (dark blue) et #F59E0B (amber). Bâtiments avec vitrines de commerces. Animaux零. Style Remotion: formes geometriques simples, ombres douces, palette restrainte. Format: SVG.
```

### Scene 3 (13-18s) - 3 vs 25 avis
```
Deux ecrans de smartphone côte à côte. Gauche: fiche Google avec "3 avis" en rouge. Droite: fiche avec "25 avis" en vert avec etoiles. Gros numbers. Style: icônes plates, couleurs contrastées, fond sombre. Format: SVG.
```

### Scene 4 (19-24s) - 67% clics
```
Barre de recherche Google stylisée. 3 premiers résultats entourés d'un halo glow vert. Pourcentage "67%" en grand avec animation counter. Style: minimal, fond sombre, accent vert #10B981. Format: SVG.
```

### Scene 5 (25-30s) - Commerçant inquiet
```
Portrait stylisé d'un commerçant (silhouette ou illustration plate) avec expression souciee. Couleurs: tons orange/jaune warm. Style: vectoriel simple, regard vers le bas. Format: SVG.
```

### Scene 6 (31-36s) - Solution
```
Ecran lumineux avec logo Expert Local. Fond: dégradé de #1E293B vers #10B981 (dark vers vert). Icône: fleche vers le haut ou checkmark. Texte: "On résout ça". Style: moderne, glow effect. Format: SVG.
```

### Scene 7 (37-44s) - 3 étapes
```
Infographie 3 cercles connectés horizontalement:
1. "Analyse" - icône loupe
2. "Scripts" - icône bulles dialogue
3. "Suivi" - icône graphique montant

### Scene 8 (45-50s) - Témoignage
```
Carte de témoignage avec: photo de profile (cercle), note 5 étoiles dorées, texte "Marie, Restauratrice". Badge "10 avis en 10 jours" en vert. Style: glassmorphism, fond flou, couleurs amber/dark. Format: SVG.

Graphique style "up only" - courbe quimonte剧烈. Axe Y: "+240%". Axe X: "30 jours". Zone sous la courbe en vert transparent avec glow. Style: minimal, axes fins, zone fill gradient. Format: SVG.

### Scene 10 (59-65s) - Diagnostic gratuit
```
Mockup de formulaire avec champs: Nom commerce, Email, Telephone. Bouton "Diagnostic gratuit" en vert avec glow. Style: clean, input fields avec bordures subtiles, fond sombre. Format: SVG.

### Scene 11 (66-72s) - CTA bouton
```
Gros bouton "Obtenir mon diagnostic" avec animation pulse (scale subtle). Fond avec subtle radial gradient. Style: bouton premium, shadow longue, hover effect visible. Format: SVG.
```

### Scene 12 (73-78s) - Outro
```
Logo Expert Local centré. Slogan en dessous: "On s'occupe de vos avis". Background: noir avec subtle noise texture. Style: minimal, typographie Large, finish clean. Format: SVG.

## 3. VOIX OFF - PROMPT MINIMAX

Prompt pour générer l'audio:
```
Tu es un voiceover professionnel français. Tone: positif, rassurant, expert. Pas de sentiment negatif, toujours solution-oriented.

Script:
[INSUER LE TEXTE CORRESPONDANT A CHAQUE SCENE]

Specifications:
- Format: MP3
- Duration:adapter au timing de la scene
- Style: narration claire, rythme moderé, pauses naturelles
```

---

## 4. SOUS-TITRES

Prompt pour générer les subtitles:
```
Génère les sous-titres pour une vidéo de [X] secondes.
Format: fichier SRT avec timestamps.

Texte:
[LE SCRIPT COMPLET]
```

---

## 5. ASSEMBLAGE FINAL

Outils recommandes:
- **CapCut** (gratuit, tres bon pour IA + subtitles)
- **Premiere/Final Cut** (si professionnel)
- **ffmpeg** (ligne de commande)

Etapes:
1. Importer les 12 images/SVG
2. Ajouter les transitions (fade, slide)
3. Ajouter la voix off
4. Ajouter les sous-titres (position basse)
5. Export en MP4

---

## 6. SPECS TECHNIQUES

- Resolution: 1920x1080 (16:9) ou 1080x1920 (9:16 pour reels)
- FPS: 30
- Codec: H.264
- Bitrate: 8-12 Mbps
- Audio: AAC, 192kbps
