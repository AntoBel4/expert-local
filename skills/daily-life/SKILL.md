# Skill : Daily Life

## Description
Gestion du quotidien : résumé matinal, budget, courses, journal, veille prix.
Partageable avec la femme d'Antoine.

## Commandes
- "Résumé matinal" → météo Chartres + agenda + emails urgents + actu IA + news
- "Meal plan [semaine]" → menu semaine selon budget/calories/saison + liste courses triée par rayon
- "Journal vocal" → transcrire notes vocales et compiler un journal structuré du soir
- "Veille prix [produit] [seuil]" → surveiller prix Amazon, alerter sous le seuil
- "Comparateur [produit]" → comparer prix Amazon, Cdiscount, Fnac
- "Inbox zero" → trier emails, supprimer spam, classer par urgence, brouillons de réponse
- "Docs admin" → gérer /impots, /assurances, /immobilier, /santé avec dates d'expiration

## Briefing matinal (cron 7h)
1. Météo Chartres (temp, pluie, vent)
2. 3 priorités du jour (depuis Notion)
3. Budget restant du mois
4. Streak actuel
5. 3 headlines news/IA

## Partage famille
Le journal vocal et le meal plan sont partageables avec la femme d'Antoine.
Format accessible, pas technique.

## Règles
- Ton chaleureux et bienveillant
- Toujours en français
- Pas de jargon technique dans les contenus partagés
