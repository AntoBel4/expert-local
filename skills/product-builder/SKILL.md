# Skill : Product Builder

## Description
Création de produits numériques générateurs de revenus.
Micro-SaaS, templates, générateurs, cours en ligne.

## Projets produits
1. Micro-SaaS générateur d'avis (white-label, 49€/mois)
2. Templates Notion payants sur Gumroad (CRM freelance, gestion locative, suivi chantier)
3. Générateur de sites vitrine CLI (JSON → site complet, vente 300€ aux artisans)
4. Cours en ligne "Maîtrisez vos avis Google" (10 modules HTML + quiz)
5. Outil d'audit SEO local freemium (lead gen)
6. Landing page generator CLI

## Commandes
- "Crée un template Notion [type]" → génère le template exportable
- "Générateur site vitrine" → script CLI qui génère un site à partir d'un JSON
- "Module cours [numéro] [sujet]" → génère un module HTML avec quiz
- "Audit freemium [nom] [ville]" → prototype de l'outil d'audit

## Stack
- HTML/CSS/JS pour les produits web
- Node.js pour les CLI tools
- Gumroad pour la vente de templates
- GitHub Pages ou VPS pour l'hébergement

## Règles
- Jamais de fausses promesses dans le marketing
- Code propre et documenté
- Toujours proposer une V1 minimale avant d'itérer
